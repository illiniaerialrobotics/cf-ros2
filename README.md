# cf-ros2
This is the software stack for autonomous CrazyFlie flight used in ECE484@UIUC. It is based off CrazySim, which uses Gazebo and crazyswarm2. 

### Set-up
Using a Python virtual environment is highly recommended. This stack has been tested on Ubuntu 22.04 + ROS2 Humble + Gazebo Garden. Other configurations may work but will not be supported officially as of SP26.
